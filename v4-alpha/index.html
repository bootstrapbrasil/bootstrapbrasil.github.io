Bootstrap é o mais popular framework HTML, CSS, e JS para desenvolvimento de projetos responsivo e focado para dispositivos móveis na web.

Baixar Bootstrap

Atualmente v4.0.0-alpha.2

Fácil de começar.

Inclua rapidamente o CSS e JS do Bootstrap em qualquer projeto. Use seu gerenciador de pacotes favorito, baixe o código-fonte, ou use o Bootstrap via CDN.

Gerenciadores de dependências

Inclue arquivos SASS e JavaScript via Bower, Composer, Meteor, ou npm. Pacotes instalados via gerenciadores de pacotes não incluem documentação, mas possuem Gruntfile e LEIAME.

Veja métodos de instalação

Bootstrap CDN

Quando você precisar incluir os arquivos compilados CSS e JS do Bootstrap, use o Bootstrap CDN, gratuito através do Max CDN folks.

Projetado para qualquer um, em qualquer lugar.

Bootstrap torna o desenvolvimento front-end web mais rápido e fácil. Ele é feito para pessoas de todos os níveis e dispositivos de qualquer forma ou tamanho.

Pré-processadores

Bootstrap usa CSS tradicional, mas seu código fonte utiliza Sass. Comece a desenvolver rapidamente com seu CSS pré-compilado ou construa com seu código fonte.

Um framework, qualquer dispositivo.

Fácil e eficiente, Bootstrap adapta seus projetos web com um simples código, de celulares para tablets e para desktops através de media queries CSS.

Muitos recursos

Com Bootstrap, você tem uma extensa e ótima documentação para elementos HTML comuns, dezenas de componentes CSS e HTML personalizados e incríveis plugins jQuery.

Bootstrap tem código aberto! É hospedado, desenvolvido e gerenciado no GitHub por pessoas como você.
